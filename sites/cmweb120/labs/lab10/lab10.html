This final lab is all about Javascript! There are three parts to this lab. The first part is simply about incorporating a 3rd party javascript snippet into the webpage. I decided to import a game. While it doesn't have any practical use for web development, I really enjoyed playing it. The moment I ran upon this cute little game on my quest to find a js snippet, I instantly knew, this was the one. I HAD to add have it in my cite. I guess one could call it love at first sight. The second part is a simple "hello world" program I wrote in javascript--my first ever program! Part three is a simple math program that I modified from an existing code provided as part of the lab.
