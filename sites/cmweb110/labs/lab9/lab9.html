A Game of Tic-Tac-Toe

In the name of Allah, the Most Gracious, the most Merciful.

This page demonstrates what you can do with javascript. Javascript allows you to do programming in a web page. The programming script you write using javascrip is executed on the client's device, not on the server.

Do note that I did not write the javascrip code in this page, as it is beyond the scope of this course. Instead, I used a sample code found in Dynamic Drive. The javascript I chose is for a simple game of tick-tack-toe.

P.S. One more thing before we move: I have added site navigation menu right below the site header (which was completd in the previous lab). Feel free to see the code to see how that was done.
