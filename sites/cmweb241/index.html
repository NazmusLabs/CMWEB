As I noted on the CMWEB 110 homepage, the assignments on this website serve as reference to the fundamental elements, based on web standards, of HTML and CSS. With each assignments being based on the work done from the previous assignments, they add one or two new concepts and features, allowing one to see a guided, piece-by-piece construction of a complex webpage. CMWEB 241 follows the same rhythm but this time with PHP and server-side scripting. The ultimate objective for most of the labs in this course is to create a fully functioning and dynamic photo gallery. Each lab builds generally builds on the PHP code from the previous lab, with the final lab resulting in the finished product.
