Developer Gold Mine

As someone who dabbles in to development be it in software or web, I cannot stress enough the value of the following resources. These three things (though not limited to) are one of the most important resources for anyone who has done or wants to do any sort of web or app development.

They are, of course, source control, code editor, and documentation. Source control (aka version control) ensures smooth collaboration with multiple developers working on a project. It helps keep track of changes to code and assets as well as making it posible to revert changes and track who is contributing what chagnes. Git and SVN are examples of version control systems. Documentations provide invaluable information and resouces about a particular software development kit, API, scripting language, dev tool, etc. They also provide code samples and examples, offer code snippits to be used in your own work, and provide technical details for the products. A code editor is at the heart of software and web development. Many popular standalone code editors exist. Although much less powerfull than a full IDE (integrated development environment) such as Eclipse and Visual Studio, these lightweight yet versatile apps' usefullness scale from individual developers to enterprise class dev teams. And, of course, a code editor is a main component of IDEs themselves!
